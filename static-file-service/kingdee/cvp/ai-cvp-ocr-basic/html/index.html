1 简介

本文档描述金蝶云苍穹视觉识别服务的相关OpenApi接口信息

2 金蝶云•苍穹 视觉识别服务 OpenAPI请求方法

2.1 调用方式

本服务API采用金蝶云•苍穹开放平台（openApi）外部可编程接口（API）调用方式

2.2 请求URL

服务域名地址/kapi/app/cvp/ocrImageInfo

列如:http://localhost:8080/ierp/kapi/app/cvp/ocrImageInfo

2.3 请求方式

  • POST

2.4 请求参数

参数名称参数必传类型
Header:
请求的消息主体编码方式Content-Type:application/json是String
用户认证后的访问令牌access_token是String
Body:
业务对象标识(苍穹的页面编码)businessObject是String
模板编码templateNumber是String
图片的Base64值imageData是String

2.5 示例

POST http://localhost:8080/ierp/kapi/app/cvp/ocrImageInfo
Request Header:
Content-Type:application/json
access_token:666522411780278272O7e12312ukdaG12338

Request Body:
{
    "businessObject": "cvp_demo",
    "templateNumber":"OPM-IDCardFront",
    "imageData":"9j/4VDuRXhpXXXXXXXXXXXXXXXXXXXXXXXXXXX"
}
